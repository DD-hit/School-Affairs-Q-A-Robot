# 校务问答机器人系统 v2.0

## 项目概述

基于原有网页的升级版本，根据调研后新增的需求进行功能增强，同时保留所有原有功能。

## 新增功能特性

### 1. 分类查找系统
- **可选流程**：默认收起，点击展开
- **8个分类**：学业相关、奖学金、宿舍生活、校园卡、图书馆、考试报名、请假申请、其他问题
- **智能提示**：点击分类自动更新搜索框提示

### 2. 答案展示优化
- **结构化展示**：分点、提纲式呈现
- **易阅读**：清晰的层次结构
- **专业格式**：符合信息展示最佳实践

### 3. 无法回答处理
- **明确提示**：清晰说明无法回答
- **解决路径**：提供多种联系方式
- **部门指引**：引导到正确的处理部门

### 4. 界面优化
- **简洁清爽**：移除冗余动画和装饰
- **无信息过载**：合理的信息密度
- **响应式设计**：适应各种屏幕尺寸

### 5. 主题系统
- **浅色/深色模式**：一键切换
- **持久化存储**：记住用户偏好
- **完整适配**：所有组件支持深色模式

### 6. 校务推送
- **分类推送**：校级/院级通知分离
- **标签切换**：点击切换不同级别通知
- **实时更新**：模拟实时通知系统

### 7. AI功能扩展
- **多模型切换**：支持不同AI模型
- **知识库导入**：自定义知识库管理
- **个性化设置**：用户偏好配置

## 使用说明

### 快速开始：
1. 双击 `index.html` 在浏览器中打开
2. 或使用本地服务器：`python -m http.server`
3. 访问 `http://localhost:8000`

### 主要操作：
1. **提问**：在搜索框输入问题或点击热门问题
2. **分类查找**：点击"展开分类"选择问题类别
3. **主题切换**：点击右上角月亮/太阳图标
4. **校务推送**：点击"校级通知"/"院级通知"切换
5. **个人中心**：点击右上角用户信息或导航栏链接
